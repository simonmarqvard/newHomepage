Discovering Trends


Node, D3.js, GoogleTrends API, RESTful API, Chart.js
Use the globe to select a country, then type in a search word to see if it has been trending. Or make a global search and the countries will light up based on the popularity of the word
This project was created for the "API - mashups" class at ITP. The assignment was to create a single page web experience that uses data from a public API and incorporates the use of at least one Javascript "interaction" library.

For this project I used the google trends API. This API allows you to see the google searches that are trending in different countries. The API has been collecting data since 2004 giving you the option to choose a startDate and endDate if you are interested in a certain time period. Also, another function provides global information displaying the countries in which your searchword was more popular.

Searching the word: "Olympics" in Brazil returns a trend in August 2016, which is when the olympics were held
The google trends library is only available through Node.js - so for this project to work I had to set up a server. I did that using Digitalocean, creating a cloud server (droplet) from which I could serve my content. I used Fetch for file transfer to the server.

The Globe was created using D3.js based on an example from Mike Bostock. The data is visualized using Chart.js and user interaction inputs/buttons were made using jQuery.
