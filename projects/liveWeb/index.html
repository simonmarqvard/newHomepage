Live-web:chat

Custom built communication application

Being a frequent user of online communication platforms like Skype, Facebook and GoogleHangouts - Live-web:chat is a project intended as a personal replacement for these services. Live-Web:chat is a communication platform that simplifies communication with other users visiting the url. Sockets.io allows online clients to see other online clients. Using peer.js and a peerserver a webRTC connection is established between the users allowing for video and audio sharing.
